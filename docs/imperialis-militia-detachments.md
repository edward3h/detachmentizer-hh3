# Imperialis Militia Detachments

* All of these detachments have the Imperialis Militia faction.
* Command Troop and Mounted Command Troop are Command units with the Officer of the Line (3) rule.
* Discipline Master Cadre is a Command unit that optionally has the Officer of the Line (3) rule, by upgrading the unit.

## Apex Detachments

### Grenadier Muster

* Slots:
  * 8 Elites. 1 is Prime.

### Ogryn Auxilia

* Slots in this detachment may only be used to select Ogryn units.
* Slots:
  * 8 Heavy Assault. 1 is Prime.

### Infantry Cohort

* Slots:
  * 8 Troops. 1 is Prime.

### Artillery Division

* Slots in this Detachment may not be used
to select Militia Medicae Units.
* Slots:
  * 8 Support. 1 is Prime.

### Cavalry Wing

* Slots in this Detachment may only be used
to select Cavalry Troop Units.
* Slots:
  * 8 Recon. 1 is Prime.

### Armoured Phalanx

* Slots:
  * 1 Lord of War
  * 7 Armour

## Auxiliary Detachments

### Oversight Delegation

* Support Slots in this Detachment may only be used
to select Militia Medicae Units.
* Slots:
  * 1 Command
  * 4 Support

### Storm Cadre

* Slots:
  * 2 Elites

### Daemonic Manifestation

* Traitor only.
* Unlocked by unit Rogue Psyker.
* Slots in this Detachment may only
be used to select Ruinstorm Daemon Brute Units
from the Daemons of the Ruinstorm faction.
* Slots:
  * 3 Heavy Assault
